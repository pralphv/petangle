## Available Scripts

#### `yarn install` 

Install all packages<br>

#### `yarn start`
###### or `npm start`

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:5000) to view it in the browser.


#### `yarn test`
###### or `npm test`
Test all spec files with Jest and Enzyme.
